# 最小浏览器回归检查清单（项目经理）

## 适用范围
- 页面入口：`src/pages/im-main.html`
- 目标：每次迭代后 5 分钟内完成核心回归确认。

## 回归步骤
1. 打开页面，确认首屏三栏布局与飞书风格样式正常。
2. 点击 `子项目A-集成协同群组`，确认会话切换到执行层。
3. 在执行层卡片点击 `查看工单详情`，确认抽屉从右侧滑出。
4. 点击抽屉遮罩与关闭按钮，确认均可关闭。
5. 观察卡片与抽屉内 SLA 倒计时，确认每秒递减并同步。
6. 切回 `项目全域总控超级群组`，确认无报错、无错位。
7. 点击 `一键穿透归因`，确认 loading 与结果卡片可渲染。

## 通过标准
- 所有步骤均通过，控制台无报错。
- 页面交互动效平滑，无明显闪烁和阻塞。

## 常见失败信号
- 抽屉无法关闭或遮罩失效。
- 倒计时停止更新或显示不一致。
- 场景切换后按钮失效。
