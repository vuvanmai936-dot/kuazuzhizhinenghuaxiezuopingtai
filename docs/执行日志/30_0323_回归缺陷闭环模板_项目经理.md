# 回归缺陷闭环模板（项目经理）

## 闭环流程
1. 发现缺陷并判级（参照 `29_0323_缺陷分级标准_项目经理.md`）
2. 输出缺陷单并派修
3. 前端提交修复说明
4. TPM 按脚本回归并给出结论
5. 关闭或打回

## 关闭标准
- 缺陷重现路径已不可复现。
- 关联功能无新增副作用。
- 文档链路完整（缺陷单、修复记录、回归结论都已落盘）。
