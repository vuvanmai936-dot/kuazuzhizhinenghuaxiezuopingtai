# 验收脚本化步骤（项目经理）

## 目标
- 用固定步骤在 5-8 分钟内完成一次标准回归验收。

## 执行脚本

### Step 0：准备
1. 打开 `src/pages/im-main.html`
2. 打开浏览器控制台
3. 清空旧日志

### Step 1：执行层切换验收
1. 点击 `子项目A-集成协同群组`
2. 验证标题、状态标签、消息卡片
3. 控制台检查是否出现异常报错

### Step 2：工单抽屉验收
1. 点击 `查看工单详情`
2. 点击遮罩关闭
3. 再次打开并点击右上角关闭
4. 重复 3 次确认稳定性

### Step 3：倒计时一致性验收
1. 在卡片与抽屉同时观察倒计时
2. 连续观察 5 秒
3. 验证两处时间同步递减

### Step 4：场景二链路验收
1. 切回总控群组
2. 点击 `一键穿透归因`
3. 验证 loading -> 结果卡片链路

### Step 5：切换稳定性验收
1. 总控/执行层来回切换 5 次
2. 验证无报错、无错位、无按钮失效
