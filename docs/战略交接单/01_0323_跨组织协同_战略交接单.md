# 战略交接单：跨组织协同场景

## 业务背景
- 当前演示目标是突出“三级群组联动 + 智能体可信协同”能力。
- 重点展示执行层的异常上报如何被智能体横向匹配并上行升级。

## 战略目标
- 3 分钟内让评审方理解系统价值：快速归因、跨组织协同、超时熔断升级。
- 对外表达口径统一为“可信数据空间最小权限 + 智能体自动校验”。

## 演示剧本要求（硬性）
- 场景一：总控层健康度总览与风险识别。
- 场景二：一键穿透归因，定位跨组织数据阻断责任链条。
- 场景三：执行层异常呼救 + 横向匹配 + 自动工单 + 超时升级提示。

## 验收标准
- 文案与角色名称需一致，不得出现“临时占位文案”。
- 场景切换需平滑，核心卡片需有明确状态与行动按钮。
- 全流程纯前端可运行，浏览器直接打开可演示。
