# 前端开发派工单：工程化迁移二期（抽屉与状态轨迹）

## 本期目标
- 在工程化入口下落地执行层增强能力：工单状态轨迹、SLA 倒计时、工单详情抽屉。

## 开发范围
1. 执行层卡片新增状态轨迹（创建/派发/待确认）。
2. 增加统一 SLA 倒计时显示，并在卡片与抽屉中同步。
3. 点击“查看工单详情”打开右侧抽屉，支持遮罩与关闭按钮关闭。
4. 抽屉包含：工单信息、可信数据空间校验记录、升级规则。
5. “转人工提报 / 补充现场日志”增加组长权限态标识。

## 验收标准
- 详情抽屉连续开关 5 次无异常。
- 倒计时每秒刷新，卡片与抽屉展示一致。
- 从执行层切回总控层后无报错。
- 页面保持纯前端可直接运行。
