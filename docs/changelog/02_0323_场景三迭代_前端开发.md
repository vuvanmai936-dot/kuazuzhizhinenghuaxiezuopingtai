## 本次改动
- 新增左侧会话点击切换逻辑：支持从总控群切换到“子项目A-集成协同群组”。
- 新增平滑切换动画：聊天主区在会话切换时执行 Fade out / Fade in。
- 切换后更新 Header 上下文：
  - 标题：`子项目A-集成协同群组`
  - 状态：`一线作业与异常上报节点`
- 渲染执行层默认消息流：
  - 应用方-张三上报 `8080 端口不通` 异常消息。
  - 执行辅助智能体返回 Adaptive Card 响应卡。
- 响应卡新增关键内容：
  - 异常分级：跨组织资源对接异常（较重）
  - 横向匹配：建设方-李工

- 响应卡新增操作按钮：
  - `查看工单详情`
  - `转人工提报 / 补充现场日志`
